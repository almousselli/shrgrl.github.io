Merhaba arkadaşlar. Bugünkü yazımda sizlere Rest API, Node.js gibi kavramlardan bahsederek beraber basit bir Rest API örneği geliştirip, bu örneğimizi HEROKU platformuna nasıl yükleyeceğimizi göreceğiz.

Uygulama yazmaya başlamadan önce uygulamayı çalıştırabilmek için gerekli ortamlara sahip olmamız gerekli. Bunun için ilk olarak eğer bilgisayarımızda yoksa Node.js indirmemiz gerek. Node.js, genelde JavaScript ile yazılan komutların sunucu tarafında çalışmasını sağlayan, ağ bağlantılı uygulamalar için geliştirilmiş bir çalıştırma ortamıdır. Bilgisayarımızda node.js’i edinmek için kendi sitesinden kolayca indirebiliriz.


{% highlight language %} //Gerekli harici express modülü için var express=require('express'); var app=express(); app.get('/',function(req,res){ //bu metot http kök dizinine istek gönderiyoruz res.send('Hello World'); //hello world yanıtını alıyoruz }); app.listen(8080,function(){ //uygulamamız localhost:8080 portunda çalışacaktır console.log('Web Server is running at http://127.0.0.1:8080/'); }); {% endhighlight %}

NodeJS ile web uygulama geliştirmek için Express modülünü kullanıyoruz. Bu modül uygulamamızda kullanacağımız “get” gibi metotlar için gereklidir. Uygulamayı kendi kök dizininde command promt açarak çalıştıracağız.

İlk olarak package.json dosyasını oluşturmak için aşağıdaki komutu çalıştıralım.

{% highlight language %}$ npm init{% endhighlight %} Daha sonra kodda kullandığımız harici modülleri yani express modülümüzü indirelim. {% highlight language %}$ npm install express -s{% endhighlight %} Harici indirme işlemlerini tamamladıktan sonra uygulamamızı çalıştırabiliriz. {% highlight language %}$ node helloworld.js{% endhighlight %}
